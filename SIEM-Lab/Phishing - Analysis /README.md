# Phishing Email Investigation

## Overview
This project demonstrates the investigation of suspicious emails to determine whether they are phishing attempts. The analysis focuses on identifying malicious indicators within email headers, domains, and embedded links.

## Tools Used
- MXToolbox
- VirusTotal
- Email Header Analyzer

## Investigation Process
Suspicious email samples were analyzed to identify indicators of compromise and validate the authenticity of the sender.

Analysis activities included:

- Extracting and analyzing email headers
- Verifying SPF, DKIM, and DMARC authentication results
- Identifying spoofed sender domains
- Extracting embedded URLs and attachment hashes
- Cross-checking indicators with threat intelligence platforms

## Threat Indicators Identified
The following phishing indicators were detected:

- Domain spoofing
- Suspicious sending infrastructure
- Malicious URLs embedded in email body
- Attachments linked to known malicious hashes

## Outcome
This project helped develop practical SOC investigation skills including:

- Email threat analysis
- Phishing detection techniques
- IOC extraction
- Threat intelligence verification
