# Network Traffic Analysis (PCAP Investigation)

## Overview
This project focuses on analyzing captured network traffic (PCAP files) to identify suspicious activity and understand attacker behavior within a network.

## Tools Used
- Wireshark
- Zeek
- Nmap

## Investigation Process
Captured network packets were analyzed to identify abnormal traffic patterns and possible indicators of compromise.

Analysis activities included:

- Inspecting packet streams
- Filtering suspicious traffic
- Identifying unusual protocols and ports
- Reconstructing communication sessions

## Threat Indicators Detected
During the investigation, the following suspicious activities were identified:

- Port scanning attempts
- ARP spoofing patterns
- Unencrypted credential transmission
- Possible command-and-control beaconing traffic

## Outcome
This investigation helped develop practical skills in:

- Packet analysis
- Network traffic monitoring
- Identifying malicious communication
- Understanding attack patterns in network traffic
