# SIEM Detection Lab

## Overview
This project demonstrates the deployment of a basic Security Information and Event Management (SIEM) environment to monitor endpoints and detect suspicious activity.

## Tools Used
- Wazuh SIEM
- Splunk Cloud
- VirtualBox
- Ubuntu Server
- Windows 10
- Kali Linux

## Lab Setup
A home lab environment was created using VirtualBox where multiple virtual machines were deployed and connected to a centralized SIEM system.

Endpoints monitored:
- Windows 10 machine
- Kali Linux attacker machine

Logs from these systems were collected and analyzed using Wazuh and Splunk.

## Attack Simulations
The following attack scenarios were simulated in the lab environment:

- Brute force login attempts
- Network reconnaissance scans
- File integrity monitoring alerts

## Detection Process
Security alerts generated by Wazuh were analyzed and correlated in Splunk dashboards. Detection events were mapped to relevant MITRE ATT&CK techniques to understand attacker behavior patterns.

## Outcome
This lab helped develop practical experience with:

- Security event monitoring
- Alert triage
- Log analysis
- Attack detection
- SOC investigation workflows
